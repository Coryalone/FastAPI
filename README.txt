Добрый день!
Мое приложение реализует методы RESTful API для управления встречами. С возможностью осуществить HTTP-запрос к соответствующему методу RESTful API через Cli обертку

Для того, чтобы развернуть приложение, я рекомендую использовать среду разработки PyCharm


Итак, запускаем IDE PyCharm. Открываем проект. Настраиваем окружение, в терминале прописываем:
pip install FastAPI 
pip install uvicorn  
pip install sqlalchemy


для запуска сервера нужно ввсти команду в терминале:
 uvicorn maybe:app --reload

для работы с cli через cmd глобально устанавливаем:
python pip install Typer
python pip install requests

Для того, чтобы прочитать все через cli нужно выполнить следующую команду:
пример 

python cli.py read http://127.0.0.1:8000 0 5

python cli.py read 1-й аргумент это host, 2-й аргумент - это смещение, 3-й это лимит нужно указать 2 цифры
возвращает выборку по указаным критериям в формате json


Для того, чтобы прочитать данные о встрече, по id через cli нужно выполнить следующую команду:
пример

python cli.py read-by-id http://127.0.0.1:8000 11

1-й аргумент это host, 2-й id нужно указать  цифру
возвращает конкретную запись в формате json


Для того что бы передать данные постом через cli нужно выполнить следующую команду:
пример

 python cli.py write http://127.0.0.1:8000 "{\"name\":\"Salary\",\"created_at\":\"2022-02-21T17:50:13\",\"finished_at\":\"2022-02-21T17:55:13\",\"members\":\"Ivan@mail.ru, Ivan@mail.ru, Ivan@mail.ru\"}"

 python cli.py write 1-й аргумент это host, 2-й аругмент это json данные 

возвращает уникальный индификатор в формате json


Для того что бы изменить данные путом через cli нужно выполнить следующую:  
пример

python cli.py update_record http://127.0.0.1:8000 17 "{\"name\":\"Nastolki\",\"created_at\":\"2022-02-21T17:50:13\",\"finished_at\":\"2022-02-21T17:55:13\",\"members\":\"Ivan@mail.ru, Ivan@mail.ru, Ivan@mail.ru\"}"

1-й аргумент это host, 2-й id встречи, 3-й аругмент это json данные

возвращает изменненные данные в формате json


Для того, чтобы удилить встречу по id, через cli нужно выполнить следующую команду:
пример

python cli.py delete http://127.0.0.1:8000 7

1-й аргумент это host, 2-й id нужно указать 1 цифру 
удаляет конкретную запись
